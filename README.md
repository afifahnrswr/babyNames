# babyNames
Get a list of filtered baby names from a csv file

Example Use:
```
gary@laptop ~/blog/babyNames $ ./babyNames.py 
gary@laptop ~/blog/babyNames $ head names.csv 
Emma,1,20799
Ava,2,15586
Mia,3,13442
Ella,4,8489
Hannah,5,6512
Anna,6,5639
Leah,7,5563
Nora,8,4708
Sarah,9,4647
Stella,10,4165
```
